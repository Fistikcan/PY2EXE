# 🐍 PY2EXE

**A modern and user-friendly graphical interface for PyInstaller.**  
Say goodbye to the command line and turn your Python scripts into `.exe` files in seconds.

---

## 🚀 What is PY2EXE?

PY2EXE is a free tool designed to convert your Python projects into **standalone Windows executable files (.exe)** or **ready-to-distribute folder structures**. It's built on top of the PyInstaller engine, but you no longer need to memorize long commands to use it. Everything happens through a sleek, dark-themed, and fluid interface.

Whether you're looking to share a small script or prepare a large project for end users, PY2EXE lets you visually configure all your build settings with ease.

---

## ✨ Features

- **Drag & Drop** – Simply drag your Python file onto the window or select it with one click
- **All PyInstaller Options** – Output format, console mode, includes/excludes, Windows/macOS settings, icon, UPX compression, and much more
- **Section-Based Layout** – Options are grouped logically so you can find what you need instantly
- **Live Progress Bar** – Track each stage of the build in real time (Analyzing → Processing → Packaging → Building EXE → Done)
- **Terminal Output (Optional)** – Hide the build logs and enjoy a clean loading screen if you prefer
- **Turkish & English** – Switch languages instantly with a single button
- **Modern Interface** – Dark theme, smooth animations, and smart tooltip boxes
- **Automatic PyInstaller Installation** – If PyInstaller isn't already on your system, the app downloads it for you on first launch

## 🛠️ Usage

1. **Launch the application.**  
   Just run `python py2exe.py` in your terminal.

2. **Select your Python file.**  
   You can drag and drop it or click the "Select File" button.

3. **Configure your build options.**  
   Check all the settings you need – output format, console preference, data files to include, icon, and more.

4. **Start the build.**  
   Click the "Start Build" button and wait for the process to complete.

5. **Get your output!**  
   Once the build finishes, your `.exe` file or folder will be ready in the location you specified.

## ⚙️ Some Supported Options

All of them are waiting for you in the interface, complete with descriptions.
